# workflows/
> L2 | 父级: /agents/AGENTS.md

Playbook workflow contracts for the 30x Outreach Agent. Workflows bind role
surfaces to overlay bindings, approval gates, evidence packets, readbacks, and
proactive GEB learning routes. They are path-portable contracts, not runtime logs.

成员清单

thirtyx-outreach-run-outreach-experiment.workflow.md: Preview-first workflow that builds verified audience/campaign contracts and dual-gated approval packets without provider writes.
thirtyx-outreach-observe-and-generate-next-wave.workflow.md: Learning workflow that converts aggregate analytics into a deterministic decision and one-variable next-wave brief.

边界
workflows/ owns process shape. The 30x skill owns deterministic execution logic;
the role owns reusable Outreach principles, the overlay owns tenant/provider
bindings, and state/ owns semantic readbacks rather than experiment truth.
